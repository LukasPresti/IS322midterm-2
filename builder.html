What is this tool?

This voice-to-website POC allows you to instantly generate and deploy front-end web pages using only your voice. It leverages OpenAI's Whisper for transcription, an LLM for UI code generation, and the GitHub API to deploy your newly created site live to GitHub Pages.

How to use properly:

  1. Click Start Recording and clearly describe the layout, styling, and content of the site you envision.
  2. Click Stop & Build. The tool will process your audio and construct a functional webpage.
  3. Monitor the Build Logs to track progress and view the Live HTML Preview on the right.
  4. Upon success, use the provided link to view your newly deployed web application!
